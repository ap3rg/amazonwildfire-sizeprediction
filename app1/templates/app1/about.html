AWPS - Amazon wildfire prediction system

This project was design for Machine Learning class at la Universidad de los Andes in Colombia. Three machine learning models were trained using fire data from NASA's VRII satelite system, (resolution 375m) and meteorolical data from NASA's Global FWI. It aims to predict burnt area based on weather and dryness index.
